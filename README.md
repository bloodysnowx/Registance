Registance
====
# Overview

# How To Develop

```
git clone git@github.com:bloodysnowx/Registance.git
cd Registance
brew install typesafe-activator
brew install heroku
git remote add heroku https://git.heroku.com/bloodysnowregistance.git
```

# How To Run
activator run

# How To Run as Release
activator start
or
activator clean stage
